## Descripción

Esta es una aplicación web que te permite calcular cuánto “te cuesta” realmente un producto o servicio en términos de tu tiempo de trabajo.

En lugar de pensar solo en el precio en euros, la herramienta convierte esa cantidad en horas de trabajo necesarias para pagarlo, teniendo en cuenta tu salario y tu jornada laboral.

El objetivo del proyecto es transformar una cifra abstracta (salario mensual o anual) en algo tangible: el valor real de cada hora trabajada y el coste real de una compra en tiempo de vida invertido.
